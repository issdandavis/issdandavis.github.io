Sacred Tongue Encoder

A tongue is one of six constructed languages (KO, AV, RU, CA, UM, DR), each tied to a governance domain — every byte of your input is mapped to a unique syllable token in the chosen tongue, and the tongue's golden-ratio weight sets the cost of operating in that domain.

Type text, pick a tongue, see it encoded in real time. Each byte maps to a unique token in the selected conlang.

How it works: Each Sacred Tongue has a 256-token vocabulary — one token per byte value. Text is UTF-8 encoded, then each byte maps to a conlang token. The tongue's phi-weighted cost (1.0 for KO up to 11.09 for DR) determines how expensive governance operations are in that domain. Higher-privilege tongues cost exponentially more, making unauthorized access mathematically infeasible.

Try it: Switch between tongues and watch the same text encode differently. Copy tokens from the output, paste them in the decode box — they decode back perfectly.
