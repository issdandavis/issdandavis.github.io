👋 About Me

Hey, I'm Issac. I'm a systems architect from Port Angeles, Washington.


I build SCBE-AETHERMOORE — an AI governance framework where attacks get priced out by geometry instead of blocked by classifiers. One formula: H(d,R) = R². Deeper and wider attacks cost exponentially more. Simple idea, hard math, running code.


I wrote a novel about it because I needed people to remember the architecture after they closed the docs. The Six Tongues Protocol — 22 chapters, 70K words, same vocabulary as the codebase.


I don't code directly. AI writes the code, I architect the systems and make the decisions. My daily driver is a Chromebook with 2.7GB of RAM. The heavy stuff runs on HuggingFace and GitHub Actions. You don't need a fancy machine to build real things.


When I'm not building, I'm probably researching hyperbolic geometry, reading about post-quantum crypto, or painting Bob Ross style landscapes with AI.
